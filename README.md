# cig-backoffice

## Dependencies

```
NodeJS >= 14.17.3
Yarn >= 1.22.1
```

## Setup

Clone the project.

```console
$ git clone git@github.com:edumoreira1506/cig-backoffice.git
$ cd cig-backoffice
```

Install all packages.

```console
$ yarn
```

For run server local.

```console
$ yarn start
```

## Tests

For run tests.

```console
$ yarn test
```

## Linter

For run eslint.

```console
$ yarn lint
```
